Klientská část

Tato část má za úkol zobrazovat uživateli to, co vidí a co možná mu nejvíc zpříjemnit pobyt na stránce. Dále pak dynamicky mění stránku za chodu.

Na začátku se uživateli načte primitivní HTML stránka, která si k sobě asynchroně dotáhne balík javascriptu, který si celý provoz stránky vezme na starost. Například jediný HTML tag na této stránce je v základu tag <app-root>.

Se servrovou částí komunikuje tak, že data získáná od uživatele zformátuje do JSON formátu a pošle je zpět na server, kde to nějaký server (nejčastěji nginx) přeproxuje na servrovou část. Jelikož se volání provádí asynchroně, tak jede aplikace pořád svižně.
